# Aprendizados coletivos da Hurtz

Aprendizados que qualquer gestor descobriu e que vale propagar pra todos.

## Formato dos arquivos

Um arquivo por mês: `learnings-YYYY-MM.md` (ex: `learnings-2026-05.md`).

Skill `aprender-com-cliente` escreve aqui automaticamente quando o gestor diz "aprendi que...".

## Formato de cada entrada

```markdown
## YYYY-MM-DD — {Título curto do aprendizado}
**Autor:** @{slug-gestor}
**Contexto:** {situação onde foi observado, qual cliente, qual tipo de campanha}
**Aprendizado:** {o que foi descoberto}
**Aplicar quando:** {gatilho — em que situações futuras isso é útil}
**Confiança:** {alta / média / hipótese — vale validar mais}
```

## Princípio

Aprendizado coletivo é o que diferencia uma equipe que aprende junto de 3 gestores trabalhando em silos. Sempre que descobrir algo replicável, registre.

Se for específico de UM cliente (não vale pra outros), vai pra `clientes/{X}/aprendizados.md`, não aqui.
