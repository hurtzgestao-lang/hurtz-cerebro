# Playbooks — Hurtz Company

Rotinas e respostas pra situações específicas. Diferente de "metodologia" (processo geral), playbook é "o que fazer quando X acontecer".

## Conteúdo (cresce ao longo do tempo)

- `escalada-cliente-em-risco.md` — sinais de churn, diagnóstico, intervenção, escalada pro Klebson
- `escalada-cpl-alto.md` — quando CPL sobe inesperadamente, passos de diagnóstico
- `kickoff-cliente-novo.md` — fluxo de onboarding de cliente recém-fechado
- `fechamento-mensal.md` — relatório mensal pro cliente, dados a coletar, formato
- `reuniao-gestores.md` — pauta padrão da reunião semanal de gestores

## Princípio

Playbook resolve dor recorrente. Sempre que um problema acontecer pela 3ª vez, vira playbook.
