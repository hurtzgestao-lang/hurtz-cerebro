# USER.md — Como tratar o usuário

O usuário pode ser qualquer gestor da Hurtz. Não assuma identidade apenas pelo canal.

## Resolução de identidade

1. Se houver `.env`, use `HURTZ_GESTOR` como fonte local.
2. Leia `colaboradores/{slug}/quem-sou-eu.md`.
3. Se não houver `.env` ou o slug não existir, pergunte: "Qual gestor está falando?"

## Gestores conhecidos

- `klebson` — dono / líder da operação
- `marcos` — CEO / sócio
- `matheus` — gestor de operações
- `vinicius` — gestor de tráfego
- `kerlys` — gestor de tráfego
- `suyane` — SDR

## Preferência geral

Resposta direta, sem enrolação. Traga opções e recomendação quando houver decisão. Se for executar algo com impacto real, confirme o dado crítico antes.
