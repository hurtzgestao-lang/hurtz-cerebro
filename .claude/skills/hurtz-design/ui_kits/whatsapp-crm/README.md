# WhatsApp CRM Lead Cards

Card de lead enviado pelo SDR ao representante de consórcio via WhatsApp. Padronizado, com ficha completa e botão de ação.

## Variações
- **Quente** (verde Resultado #1A7A4A) — pronto para fechar
- **Morno** (Brasa Viva #C06018) — em follow-up
- **Frio** (azul Informação #3A6FBF) — primeira abordagem

## Anatomia
1. Tag de temperatura (caps, peso 700, +12% tracking)
2. Nome do lead (Bold, 14 px)
3. Status curto (Regular, Grafite)
4. Grid 2×2: Cota · Categoria · Origem · Cidade
5. CTA Brasa Viva: "ABRIR FICHA COMPLETA →"
