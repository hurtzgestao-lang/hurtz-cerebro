# Meta Ads UI Kit

Criativos quadrados (1:1) para campanhas Meta Ads. Três templates cobrem 80% dos casos:

1. **Light · Headline + CTA** — fundo off-white, oferta direta, número grande, CTA Brasa Viva.
2. **Dark · Case** — fundo carvão, número de impacto em destaque com `M` em Brasa, prova social.
3. **Stat grid · Prova de operação** — quatro métricas em grid, sem headline.

## Regras

- Fundo: off-white (#F5F2EC) **ou** carvão (#181614). Nunca branco puro.
- Logo no topo, versão completa, altura ≤ 24 px.
- Headline ≤ 2 linhas, Inter Bold.
- CTA Brasa Viva (#C06018), texto branco, retângulo 4 px.
- Máximo 2 cores por criativo (não conta o off-white/carvão de fundo).
